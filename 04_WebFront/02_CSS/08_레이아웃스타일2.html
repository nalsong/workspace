요소의 여백(영역) 관련 속성

        HTML요소는 총 4가지의 영역으로 구성되어 있다.

        1. content영역
            -요소의 내용이 작성되는 영역
            (시작태그 ~ 종료 태그 사이에 작성되는 내용을 생각하면 된다.)
        2. padding영역
            - content영역과 border영역 사이
        3. border영역
            - 요소의 테두리가 지정되는 영역
            - content보다 바깥쪽에서 content영역을 감싸고 있음
        4. margin영역
            - 다른 요소와의 간격을 나타내는 영역

padding, border, margin의 방향성

        padding, border, margin은 방향에 따라 크기를 별도 지정할 수 있다.

        (방향은 상, 하, 좌, 우 4개만 존재)

        - 영역 이름 : 값1개;
            -> 상, 하, 좌, 우 모두 동일한 크기 지정
            ex) padding: 30px; margin: 50px; border: 10px;
        - 영역 이름 : 값1 값2;
            ->  값1 : 상하, 
                값2 : 좌우
        - 영역 이름 : 값1 값2 값3;
            ->  값1 : 상
                값2: 좌우
                값3: 하
        - 영역 이름 : 상 우 하 좌 (상부터 시계방향)


        - top, bottom, left, right를 이용하는 방법
        영역이름-top    : 값;
        영역이름-bottom : 값;
        영역이름-left   : 값;
        영역이름-right  : 값;
